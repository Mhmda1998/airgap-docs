# AirGap Docs

This repository is the official documentation and support center for **AirGap Vault** and **AirGap Wallet**. Our mission is to provide comprehensive guides to ensure the security of your digital assets using air-gapped technology.

## About the Project
This project provides detailed instructions on setting up and using cold storage solutions, ensuring that your private keys remain completely isolated from any internet-connected network.

## Documentation Structure
- 🛡️ **airgap-knox**: Documentation regarding the protection of isolated systems.
- 🔐 **airgap-vault**: Setup and security guidelines for the Vault.
- 📱 **airgap-wallet**: A guide for using the wallet and managing assets.
- 🔄 **account-recovery**: Steps for recovering your account in case of device loss.
- ❓ **FAQ**: Answers to frequently asked questions.
- 🚀 **UPCOMING**: Updates and features currently under development.
